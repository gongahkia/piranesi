# `Piranesi` webapp

## Web build

1. Available at [`https://piranesi-front.vercel.app/`](https://piranesi-front.vercel.app/).

## Local deployment 

1. First run.

```console
$ cd piranesi
$ npm install @tanstack/react-query
$ npm run dev
```

2. Then visit [`http://localhost:3000`](http://localhost:3000).
